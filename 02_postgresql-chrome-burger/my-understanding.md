PostgreSQL คือ คือระบบจัดการฐานข้อมูลเชิงสัมพันธ์ ใช้สำหรับเก็บ จัดการ และดึงข้อมูลโดยใช้ภาษาSQL
Server = ตัวระบบ PostgreSQL
Database = ฐานข้อมูล
Schema = กลุ่มของ Table
Table = ตารางเก็บข้อมูล
 Row = แถวของข้อมูล
 Column ฟิลด์ของข้อมูล
 
 Data Types 
 INTEGER = ตัวเลข
 VARCHAR = ข้อความ 
 TEXT = ข้อความยาว 
 BOOLEAN = true / false 
 DATE = วันที่ 
 TIMESTAMP = วันและเวลา 
 JSON = เก็บข้อมูล JSON 

PRIMARY KEY ของแต่ละตารางจะเชื่อมกับ FOREIGN KEYของอีกตาราง

ลำดับการเขียนQuery
SELECT = เลือก column ที่จะแสดง
FROM = เลือกตาราง
WHERE = กรองข้อมูล
GROUP BY = จัดกลุ่มข้อมูล
HAVING = กรองกลุ่ม
ORDER BY = เรียงข้อมูล
LIMIT = จำกัดจำนวนข้อมูล
